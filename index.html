HTML Cheat Sheet

Syntax Description
HTML <html></html> Creates a html document
HEAD <head></head> title and other info not displayed
BODY <body></body> visable portion of document
TITLE <title></title> name of document in title bar
EM <em></em> Emphasizes a word in italics
STRONG <strong></strong> Emphasizes a word in bold
PARAGRAPH <p></p> Creates a new paragraph
BREAK <br> Inserts an in line break
BLOCK QUOTE <blockquote></blockquote> puts content in a quote
UNORDERED LIST <ul></ul> Creates an unordered list
ORDERED LIST <ol></ol> Creates an ordered list with numbers
LIST <li></li> enters a list item
